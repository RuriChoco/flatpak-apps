Install via GNOME Software

This will download a 1-click install file. Open it with GNOME Software to instantly add the repository and install Photopaper in one go!

Manual Installation
Add the Repository
Register the Flatpak remote on your system.
flatpak remote-add --if-not-exists RuriChoco https://rurichoco.github.io/flatpak-apps/RuriChoco.flatpakrepo
Install App
Download and install Photopaper.
flatpak install RuriChoco com.github.RuriChoco.Photopaper
