RuriChoco Apps

The official Flatpak repository for my personal applications.

Add to GNOME Software (Graphical)

The easiest way to install apps is to add the repository directly to your software center. Once added, you can search for and install apps like Photopaper graphically.

⬇️ Add Repository to Flatpak

Command Line Installation

  1. Add the repository to your system:

    flatpak remote-add --if-not-exists RuriChoco https://rurichoco.github.io/flatpak-apps/RuriChoco.flatpakrepo

  2. Install Photopaper:

    flatpak install RuriChoco com.github.RuriChoco.Photopaper
